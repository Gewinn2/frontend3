## Запуск и установка
**Для работы нужен установленный ЯП Go.**
1. ```git clone https://github.com/Gewinn2/frontend3```
2. ```cd practice3``` или ```cd practice4```
3. Далее нужно прописать в каждом сервере
 ```go mod tidy```
 ```go run main.go```
