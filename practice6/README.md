**Склонируйте репозиторий:**
```
git clone https://github.com/Gewinn2/frontend3.git
```

```
cd practice6
```

### Фронтенд
Перейти в каталог frontend:
```
cd frontend
```

**Установите зависимости:**
```
npm install
```

**Запустите приложение:**
```
npm run dev
```

### Бэкенд
Перейти в каталог frontend:
```
cd ../backend
```

**Установите зависимости:**
```
go mod tidy
```

**Запустите приложение:**
```
go run main.go
```
